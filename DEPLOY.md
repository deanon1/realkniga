# Деплой на Render.com

## Инструкция по развертыванию:

1. **Залейте код на GitHub**
   ```bash
   git add .
   git commit -m "Ready for deploy"
   git push
   ```

2. **Создайте аккаунт на Render.com**
   - Зайдите на https://render.com
   - Зарегистрируйтесь через GitHub

3. **Создайте новый Web Service**
   - Нажмите "New +"
   - Выберите "Web Service"
   - Подключите ваш GitHub репозиторий
   - Render автоматически найдет `render.yaml` и настроит всё

4. **Ждите завершения деплоя**
   - Render автоматически соберет и запустит проект
   - После завершения вы получите ссылку вида `https://real-kniga.onrender.com`

## Что происходит автоматически:

- Установка PHP зависимостей (composer)
- Сборка фронтенда (npm)
- Создание SQLite базы данных
- Запуск миграций
- Настройка переменных окружения

## Переменные окружения (уже настроены в render.yaml):

- `APP_ENV=production`
- `APP_DEBUG=false`
- `APP_KEY` (генерируется автоматически)
- `DB_CONNECTION=sqlite`
- `CACHE_DRIVER=file`
- `SESSION_DRIVER=file`
- `QUEUE_CONNECTION=sync`

## Бесплатный тариф Render:

- 512 MB RAM
- 0.1 CPU
- Спит после 15 минут неактивности (просыпается ~30 секунд)
- Ограничение: 750 часов/месяц

Для продакшна рекомендуется перейти на платный тариф ($7/месяц).
